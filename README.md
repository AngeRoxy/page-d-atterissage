J’ai conçu une page d’atterrissage responsive pour la boutique Wifetique en utilisant Bootstrap 5.
La navbar utilise le système responsive avec menu burger automatique sur mobile.
La section produits est organisée grâce au système de grille Bootstrap (col-lg-3 col-md-6).
J’ai utilisé des cartes Bootstrap pour afficher les produits avec un design propre et professionnel.
Des media queries et les classes Bootstrap garantissent l’adaptation à toutes les tailles d’écran.
